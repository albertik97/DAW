Las caractersticas del diseo de esta pgina son:

  • Un diseo fijo centrado con un ancho de 960 px.
  • El encabezado de nivel 1 centrado.
  • El contenido principal de la pgina se organiza en dos columnas, una de 260 px de ancho y la otra el espacio restante de 700 px.
  • La lista que explica el contenido del libro se muestra como un texto a tres columnas del mismo ancho.

Puedes modificar el cdigo HTML proporcionado para aadir los identificadores y clases que necesites. Tambin puedes aadir etiquetas <div> para definir elementos contenedores en la pgina.
